# Presentation: Meta-Learning Enhanced Protein Language Models for Fitness Prediction

## Files

- `main.tex` - Beamer presentation source
- `Makefile` - Build automation

## Compilation

### Using Make
```bash
make        # Compile PDF
make clean  # Remove auxiliary files
```

### Manual Compilation
```bash
pdflatex main
pdflatex main
```

## Requirements

- LaTeX distribution (TeX Live, MiKTeX, etc.)
- Beamer document class
- Required packages: graphicx, booktabs, tikz

## Presentation Structure

1. **Introduction** - Problem and motivation
2. **Methods** - Architecture and meta-learning approach
3. **Results** - Main results, ablations, category analysis
4. **Discussion** - Why it works, limitations
5. **Conclusion** - Summary and future work
6. **Backup slides** - Training details, statistical significance

## Slides Overview

- 16:9 aspect ratio
- Madrid theme with custom colors
- TikZ diagrams for architecture visualization
- Tables with actual experimental results
